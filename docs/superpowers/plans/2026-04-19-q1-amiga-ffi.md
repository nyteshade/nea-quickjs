# Q1 Amiga FFI — Implementation Plan

> **For agentic workers:** execute top-to-bottom, one commit per phase where
> marked. Run host verification at each gate. Follow the build-and-deploy
> procedure stored in Fina (`procedure,build,workflow`) — DON'T skip the
> `make -C library/vbcc variants` step.

**Goal:** Ship Q1 FFI at library 0.124.

**Tech:** VBCC 68k, vasm for .s, QuickJS-ng bytecode via qjsc.

---

### Phase 1 — ASM trampoline

**File:** `library/vbcc/amiga_ffi_call.s`

Single function `_qjs_amiga_trampoline(lib, lvo, regs) → ULONG`.
See spec for the exact listing.

- [ ] Write .s with `section .text,code` + `xdef _qjs_amiga_trampoline`
- [ ] MOVEM saves D2-D7/A2-A6, loads D0-D3/A0-A3 from regs, JSR indexed
- [ ] MOVEM restores, UNLK, RTS
- [ ] Register export via LVO in next phase

### Phase 2 — C surface

**File:** `library/vbcc/amiga_ffi.c`

- [ ] `qjs_amiga_openLibrary(ctx, argc, argv)` — extracts name+version, calls exec OpenLibrary, returns JSValue number (or 0)
- [ ] `qjs_amiga_closeLibrary(ctx, argc, argv)` — wraps exec CloseLibrary
- [ ] `qjs_amiga_call(ctx, argc, argv)` — extracts lib/lvo/regs object, builds `struct AmigaRegs`, calls `qjs_amiga_trampoline`
- [ ] `qjs_amiga_peek8/16/32` — direct dereferences
- [ ] `qjs_amiga_poke8/16/32` — direct writes
- [ ] `qjs_amiga_peekString(ctx, argc, argv)` — reads NUL-terminated bytes up to maxLen, returns JSString
- [ ] `qjs_amiga_pokeString(ctx, argc, argv)` — writes JSString bytes + NUL
- [ ] `qjs_amiga_allocMem(ctx, argc, argv)` — wraps exec AllocMem
- [ ] `qjs_amiga_freeMem(ctx, argc, argv)` — wraps exec FreeMem
- [ ] `qjs_amiga_makeTags(ctx, argc, argv)` — iterates JS array, allocates TagItem memory, writes pairs + TAG_DONE terminator
- [ ] `QJS_InstallAmigaFFIGlobal(base, ctx)` — registers all the above on `globalThis.__qjs_amiga_*` using `JS_NewCFunction` + `JS_SetPropertyStr`
