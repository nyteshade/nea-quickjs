A quick compile of Andrew Tridgell's rzip for the 68k Amiga platform. The
library is a very capable compression library in the line of bzip2. This
one was compiled without specifying -noixemul, so presumably it requires
this library.

The version of ixemul.library on my system is 48.0 in the case that you
need to source the library.
